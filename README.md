## Getting Started
Clone respository:
```git
    git clone https://github.com/its-nedum/proexe.git
```

Install dependencies:
```bash
    npm install
```

Start React server
```bash
    npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the website.
